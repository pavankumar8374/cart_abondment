We designed and developed an AI-powered cart abandonment recovery system that detects when users
leave items in their cart and automatically sends personalized WhatsApp reminders using the Twilio API. The
project integrates conversational AI to engage customers in real-time, answer product-related queries, and
encourage them to complete their purchases. To simulate the e-commerce environment, I built a responsive
frontend with HTML/CSS/JavaScript and a Flask backend to track user activity and trigger automated messages.
